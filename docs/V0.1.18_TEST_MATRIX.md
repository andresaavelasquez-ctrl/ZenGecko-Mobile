# Zen Browser v0.1.18 — Matriz de validación

## Dirección y búsqueda
1. Mantener pulsado el texto de la dirección.
2. Probar Seleccionar, Seleccionar todo, Copiar, Cortar y Pegar.
3. Confirmar Buscar, Ir, Hecho y Enter.
4. Revisar sugerencias y recientes en Noche y Día.

## Tema web
1. Abrir Google, Wikipedia, GitHub y YouTube en Noche.
2. Cambiar a Día y comprobar `prefers-color-scheme`.
3. Regresar a Noche sin perder pestañas.
4. Verificar que Gecko no use gris predeterminado.

## Ventanas
1. Abrir 20 enlaces con `target=_blank`.
2. Probar `window.open()`.
3. Probar autenticación con ventana secundaria.
4. Confirmar que no haya cierres ni popups duplicados.

## Accesos rápidos
1. Crear 20 accesos sin campo Icono.
2. Confirmar guardado inmediato.
3. Confirmar favicon automático al reabrir el panel.
4. Probar sin red y con URLs inválidas.

## Android
1. Abrir HTTPS desde otra aplicación.
2. Ver Zen en Aplicaciones predeterminadas → Navegador.
3. Solicitar el rol desde Configuración → General.
4. Probar icono circular, cuadrado redondeado y monocromático.

## Renderizado
1. Cambiar de página 50 veces.
2. Cambiar entre cinco pestañas durante cargas.
3. Minimizar y regresar durante navegación.
4. Confirmar ausencia de gris persistente.

## Release
1. Ejecutar `install.sh --release`.
2. Confirmar etiqueta `v0.1.18`.
3. Confirmar Release con `ZenBrowser-v0.1.18-debug.apk`.
