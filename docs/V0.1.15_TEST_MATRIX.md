# Zen Browser v0.1.15 — Matriz de validación

## 1. Renderizado y ciclo de vida

1. Abrir Wikipedia, YouTube y Backrooms MMO diez veces cada uno.
2. Confirmar que no quedan superficies negras, grises o blancas durante más de dos segundos.
3. Cambiar rápidamente entre cinco pestañas durante dos minutos.
4. Enviar la aplicación al fondo y regresar veinte veces.
5. Rotar el dispositivo durante carga, después de pintar y con el panel abierto.
6. Confirmar en Logcat la secuencia:
   `NETWORK_STARTED → WAITING_FOR_COMPOSITE → FIRST_COMPOSITE → FIRST_CONTENTFUL_PAINT → VISIBLE`.
7. Verificar que una recuperación de superficie no recarga la URL ni pierde formularios.

Filtro recomendado:

```bash
adb logcat -s ZenGecko/Main ZenGecko/Browser
```

## 2. Pestañas

1. Pulsar `+` rápidamente diez veces.
2. Confirmar que solo existe una pestaña vacía activa.
3. Abrir y cerrar el panel durante la creación de una pestaña.
4. Reiniciar con varias pestañas vacías antiguas y confirmar que se compactan por espacio.
5. Cambiar de pestaña y confirmar que no aparece Inicio entre páginas ya cargadas.

## 3. Descargas

1. Descargar un APK, una imagen, un PDF y un archivo grande.
2. Mantener abierto el panel Descargas y comprobar actualización en vivo.
3. Confirmar velocidad, bytes, total, porcentaje y barra dentro del borde.
4. Cancelar y reintentar.
5. Probar una imagen `data:image/...;base64,...` pequeña.
6. Probar un enlace `blob:` y confirmar el mensaje legible de incompatibilidad.
7. Confirmar que al 100 % la barra brilla y desaparece, dejando “Descarga completada”.
8. Verificar que la notificación flotante no supera 292 × 58 dp.

## 4. Menú contextual

1. Mantener pulsadas imágenes, imágenes enlazadas, GIF, video, audio y enlaces.
2. Abrir vista previa, copiar, compartir y descargar.
3. Repetir cada operación veinte veces para descartar cierres intermitentes.
4. Cerrar el menú con Atrás, rotación y cambio de aplicación.

## 5. Caché y memoria

1. Registrar almacenamiento antes de la prueba.
2. Navegar durante treinta minutos y usar vistas previas/contexto.
3. Reiniciar y dejar que se ejecute el mantenimiento automático.
4. Confirmar que `zen-assets` no supera 6 MB.
5. Confirmar que `context-media` no supera 24 MB y elimina archivos antiguos.
6. Confirmar que las capturas de transición no permanecen después de contenido visible.
7. Verificar que el mantenimiento automático no borra toda la caché web de Gecko.

## Criterio de publicación

La versión no debe publicarse como estable si aparece una superficie atascada, una pestaña vacía duplicada, una barra fuera de la tarjeta o un cierre durante las pruebas repetidas.
