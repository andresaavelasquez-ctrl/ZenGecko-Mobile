# v0.1.19 — Resource Linking Hotfix

GitHub Actions alcanzó la configuración y validación de la firma, pero AAPT2
detuvo `processDebugResources` porque `bg_landscape_tab.xml` referenciaba
`@color/zen_surface_selected` sin una definición local.

El hotfix define:

- Noche: `#211A2B`
- Día: `#EEE7FA`

La versión permanece en `0.1.19` / `versionCode 20` porque no se produjo una
APK publicable antes de esta corrección.
