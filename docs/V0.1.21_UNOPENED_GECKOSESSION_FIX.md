# Zen Browser v0.1.21 — Unopened GeckoSession Fix

## Cierre confirmado

```text
FATAL EXCEPTION: main
java.lang.AssertionError:
Must use an unopened GeckoSession instance
```

## Flujo anterior

```text
onNewSession
→ addTabInternal
→ ensureSession
→ session.open(runtime)
→ devolver la sesión ya abierta
→ AssertionError
```

## Flujo corregido

```text
onNewSession
→ crear BrowserTab
→ crear GeckoSession nueva
→ instalar delegates y ajustes
→ NO abrir
→ NO cargar URI
→ devolver GeckoSession
→ Gecko abre y carga la sesión
→ onPageStart selecciona la pestaña hija
```

## Alcance

No contiene cambios visuales ni funciones nuevas. Solo corrige el ciclo de
vida de las sesiones secundarias.
