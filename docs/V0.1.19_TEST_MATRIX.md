# Zen Browser v0.1.19 — Matriz de validación

## Identidad visual
1. Comparar la Z del launcher, panel lateral y pantalla de Inicio.
2. Probar máscara circular, cuadrada redondeada y monocromática.
3. Confirmar ausencia de esquinas negras dentro de la aplicación.

## Dirección y selección Android
1. Mantener presionada la dirección.
2. Probar doble toque sobre una palabra.
3. Probar Seleccionar, Seleccionar todo, Copiar, Cortar y Pegar.
4. Arrastrar ambos controles de selección.
5. Repetir en vertical y horizontal.

## Ventanas, redirecciones y pestañas
1. Abrir 20 enlaces target=_blank.
2. Probar window.open() seguido de dos redirecciones.
3. Confirmar que la pestaña hija queda seleccionada durante la cadena inicial.
4. Confirmar que la pestaña original permanece viva y sin recargarse.
5. Abrir el mismo enlace externo dos veces rápidamente y comprobar que no se duplica.
6. Probar inicio de sesión mediante ventana secundaria y onCloseRequest.

## Renderizado
1. Cambiar entre cinco pestañas durante cargas y redirecciones.
2. Probar 50 cambios de página sin gris persistente.
3. Minimizar y regresar durante una redirección.
4. Confirmar que no se elimina la captura anterior tras un fallo temporal.

## Panel horizontal
1. Girar a horizontal con 1, 5 y 12 pestañas.
2. Confirmar lista visible, pestaña activa, título y URL.
3. Cambiar de pestaña desde la barra fija.
4. Cambiar Personal / Trabajo / Estudio.
5. Abrir Configuración, Descargas y el panel completo.
6. Confirmar que el contenido web conserva suficiente anchura.

## Fondos Día
1. Activar Día en vertical y comprobar zen_home_day_portrait.jpg.
2. Girar a horizontal y comprobar zen_home_day_landscape.jpg.
3. Regresar a Noche y confirmar el fondo oscuro original.
4. Girar varias veces sin recrear pestañas.

## Publicación
1. Ejecutar install.sh con --release.
2. Confirmar main, etiqueta v0.1.19 y GitHub Release.
3. Confirmar ZenBrowser-v0.1.19-debug.apk.
