# v0.1.22 — Adaptive Icon Resource Hotfix

GitHub Actions falló en `processDebugResources` porque el XML usaba `@drawable-nodpi/zen_icon_v022_source`. Los calificadores de carpeta como `-nodpi` no forman parte del nombre que se usa para referenciar recursos Android.

La referencia válida es `@drawable/zen_icon_v022_source`.

Se mantiene `versionCode 23` y `versionName 0.1.22`, porque la APK anterior no llegó a generarse.
