# Zen Browser v0.1.16 — Matriz de validación

## Panel lateral
1. Abrir y cerrar el panel 20 veces.
2. Confirmar márgenes en los cuatro bordes y esquinas redondeadas.
3. Cambiar Personal → Trabajo → Estudio → Personal diez veces.
4. Verificar que el panel no se cierre al cambiar de espacio.
5. Confirmar que las sesiones web no se recargan.
6. Abrir Favoritos, Historial y Perfil desde la fila secundaria.
7. Abrir Descargas desde la esquina inferior derecha.

## Configuración interna
1. Abrir Configuración desde la tuerca inferior izquierda.
2. Confirmar que aparece una pestaña llamada Configuración.
3. Abrirla diez veces y verificar que solo exista una.
4. Cambiar opciones de cada categoría.
5. Cerrar la pestaña y volver a la página web anterior.
6. Escribir una URL desde la pestaña Configuración y comprobar que se convierte en pestaña web.
7. Reiniciar la aplicación con Configuración abierta.

## Menú contextual
1. Mantener pulsadas imágenes 20 veces.
2. Mantener pulsados enlaces 20 veces.
3. Probar una imagen enlazada, GIF, video y audio.
4. Abrir/cerrar Vista previa diez veces.
5. Abrir el menú, enviar la aplicación al fondo y regresar.
6. Confirmar que el menú inicial no descarga automáticamente una miniatura.
7. Confirmar que blob: y esquemas incompatibles no muestran Descargar/Copiar imagen.
8. Probar Copiar imagen y Compartir imagen en HTTPS.

## Búsqueda inicial
1. Tocar el lado izquierdo, centro y lado derecho de la barra grande.
2. Confirmar foco y teclado en todos los casos.
3. Usar Enter/Buscar para navegar.
4. Repetir toques rápidos y verificar que no se creen pestañas duplicadas.

## Regresión
1. Navegar por Wikipedia, YouTube y Backrooms MMO.
2. Cambiar entre cinco pestañas rápidamente.
3. Minimizar/regresar durante una carga.
4. Ejecutar una descarga completa.
5. Verificar que no aparezcan superficies negras o grises persistentes.
